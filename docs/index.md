# Agents Writing Skills

Skills and prompts for agents that write text without sounding like AI.

This site is rendered from the [knowledge base](knowledge-base.md) and [skills](skills-overview.md) in the [GitHub repository](https://github.com/11111000000/agents-writing-skills).

## What is this?

A collection of skills and prompts that help LLM agents write prose that reads as human-authored. Built on research from Aboudjem/humanizer-skill (43-pattern catalogue), harshaneel/humanize (9 levers), Wikipedia EN/RU, and academic papers (Binoculars, MASH).

## Install

```bash
git clone https://github.com/11111000000/agents-writing-skills.git
cd agents-writing-skills
./install.sh all
```

## What it is NOT

Not a tool for evading AI detection. Skills help text read as human to humans. They do not guarantee passing commercial AI detectors. See [Limitations](limitations.md).
