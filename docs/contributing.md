# Contributing

See [CONTRIBUTING.md](../CONTRIBUTING.md) in the repository root.

## Quick summary

- **Bugs:** open an issue with the bug-report template.
- **New skill:** use the [skill template](../skills/template-skill/SKILL.md), then open an issue or PR.
- **Knowledge note:** add to `knowledge/` using [`knowledge/templates/note-template.md`](../knowledge/templates/note-template.md).
- **Prompt template:** add to `prompts/` with YAML frontmatter.

Before submitting:

1. Run `./scripts/validate-skills.sh` to check frontmatter.
2. Run `./install.sh skill <name>` to test locally.
3. Run `./install.sh prompt <name>` if adding prompts.
